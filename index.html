    Get Started
    Create, deploy and host anything
    with a single command

    MDB GO is an open source deployment tool & free hosting

    To start using MDB GO / CLI install it with one command:

    npm install -g mdb-cli

    Then run:

    mdb publish

    That's it. Your project is available on the Internet.

    And yes, hosting is also free :)
